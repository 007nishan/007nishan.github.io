## About

Operations and Data Science professional with 3+ years across AI/ML product development, LLM-powered automation, full-stack web applications, and operational analytics. Currently building Gen AI-driven quality systems at Amazon's Last Mile operations — including self-improving RAG architectures, real-time dashboards, and SLA engines. Passionate about translating complex operational problems into scalable, data-driven solutions that directly prevent customer-facing defects. Background in Mechanical Engineering with an ongoing BS in Data Science from IIT Madras.

## Currently — Transportation Specialist @ Amazon

- **Resolve-It — LLM Powered Andon Cord** — Built an LLM-powered Andon Cord enabling auditors to raise SOP doubts from GeoStudio in one click.
- **Real-Time Revert Dashboard** — Built an internal web application transforming manual quality reporting into an always-current intelligence platform.
- **Slack Workflow Based Andon Cord** — Deployed a Slack-integrated anonymous doubt resolution workflow enabling auditors to raise SOP, process, and tool queries while maintaining complete anonymity.
- **AWS Hosted SLA Engine and Dashboard for Slack Workflow** — Engineered an SLA tracking engine monitoring ticket lifecycle metrics — raise-to-resolution time, escalation rates, and manager approval timelines.
- **Project Prophesy** — Built a proactive detection mechanism that flags repeat-defect addresses prone to escalation using multi-signal analysis, shifting defect handling from reactive to preventive.
