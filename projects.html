Resolve-It — LLM Powered Andon Cord

Built an LLM-powered Andon Cord enabling auditors to raise SOP doubts from GeoStudio in one click. System performs semantic retrieval against Policy 2.0 and verified responses, generates AI suggestions for POC validation, and delivers resolutions in ~5 minutes. Implements Human Feedback Reinforcement Learning — managers approve, reject, or edit responses daily, making the system self-improving. Impact: ~200 min/day recovered across 20 auditors.

Real-Time Revert Dashboard

Built an internal web application transforming manual quality reporting into an always-current intelligence platform. Managers upload daily revert case batches; system auto-identifies unique cases and computes role-aware KPIs on every page request — Pre-Appeal Quality, Post-Appeal Quality, Volume Trends, and Reason Distribution. Role-based views for auditors, managers, and skip-level leaders — eliminating ad-hoc reporting entirely.

Resolve-It 2.0 — Self-Improving GenAI Andon Cord

LMAQ GenAI Hackathon 2026 · 2026 — View hackathon idea →

Conversational RAG assistant with live case context and human-in-the-loop ticketing

Live Conversational Bot

Enhanced Resolve-It into a chatbot embedded directly in GeoStudio that loads live case context, letting auditors converse for instant, SOP-grounded guidance — making the system faster and more robust than the one-click predecessor.

Grounded RAG + Guardrails

Answers are generated only from retrieved program SOPs, anecdotes, and manager-approved resolutions, with three layers of reliability — source grounding, contextual grounding checks, and confidence thresholds that route low-certainty responses to manager review.

Human-in-the-Loop Ticketing

A “Disagree” button raises a ticket the moment the AI's answer is unsatisfactory or beyond scope; manager-resolved tickets are re-ingested into the Knowledge Base with elevated retrieval priority, creating a continuous self-improving loop.

Impact

Designed to support 300+ auditors across 3 sites served by only 14 SPOCs, recovering ~50% of SPOC bandwidth (~7 FTEs / ~14,560 hours annually) and shifting error detection from downstream QC to the point of decision.
